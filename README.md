# matrix

Matrix rain effect.

## Requirements

- Node v17.4.0
- Yarn v1.22.17

## Install

First install all the dependencies:

```sh
yarn install
```

## Usage

Then, run the development server:

```sh
yarn start
```

Open http://localhost:1234 with your browser to see the result.
